# Cold Lava Advertorial Generator

High-converting landing page builder with proven templates.

## Features

- 🎯 Multiple proven templates (SaaS, E-Commerce, Health)
- ✏️ Easy section-by-section editing
- 👁️ Live preview mode
- 📱 Mobile-responsive design
- 💾 Export to HTML
- 🎨 Cold Lava branding (cyan/orange theme)
- 🚀 Conversion-optimized layouts

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Templates Included

### 1. SaaS Product Launch
Perfect for software products with:
- Hero section with strong CTA
- Problem/solution framework
- Feature highlights
- Social proof (testimonials)
- Money-back guarantee
- Final CTA with urgency

### 2. E-Commerce Product
Optimized for physical products with:
- Compelling headline
- Pain point identification
- Product benefits
- Customer testimonials
- Satisfaction guarantee
- Limited-time offer CTA

### 3. Health & Wellness
Ideal for supplements/health products with:
- Natural solution positioning
- Trust-building (doctor recommendations)
- Benefit-focused features
- Real testimonials
- Special offer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Usage

1. Choose a template that fits your product category
2. Edit each section's content (headlines, body text, CTAs)
3. Preview the full landing page
4. Export as HTML file for deployment

## Section Types

- **Hero**: Main headline and primary CTA
- **Problem**: Identify customer pain points
- **Solution**: Present your product as the answer
- **Features**: List key benefits and features
- **Testimonials**: Build trust with social proof
- **Guarantee**: Reduce risk with money-back guarantees
- **CTA**: Final conversion push with urgency

## Conversion Best Practices

Each template is built with proven conversion principles:
- Clear value proposition
- Problem-solution framework
- Social proof placement
- Multiple CTAs throughout
- Urgency and scarcity elements
- Trust signals (guarantees)
